Falta um passo: conectar ao Firebase

Seus anúncios serão salvos na nuvem para que todas as famílias vejam. Leva ~3 minutos e é gratuito.

  1. Acesse o console do Firebase e clique em Adicionar projeto.
  2. No menu lateral, abra Build → Firestore Database e clique em Criar banco de dados (pode iniciar no modo de teste).
  3. Abra Build → Authentication → Sign-in method e ative o provedor Anônimo.
  4. Em Configurações do projeto ⚙️ → Seus apps, registre um app Web e copie o objeto firebaseConfig.
  5. Cole esse objeto neste arquivo, dentro do bloco marcado COLE AQUI, salve e recarregue a página.

Regras de segurança recomendadas (cole em Firestore → Regras):

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /livros/{id} {
      allow read: if true;
      allow create: if request.auth != null
                    && request.resource.data.ownerId == request.auth.uid;
      allow update, delete: if request.auth != null
                    && resource.data.ownerId == request.auth.uid;
    }
  }
}
